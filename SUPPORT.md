# Support

## For Engine AI clients

If you're a client and you've hit an issue:

1. Check the specific project's `README.md` and `docs/` folder first.
2. Post in the shared channel we set up for your engagement.
3. For urgent issues (down in production, data loss, billing), message a founder directly.

## For internal issues

Open an issue in the relevant repo using the bug template. Tag severity honestly. S1 gets a founder's attention within the hour.

## For outside contributors

Open an issue. Tag a founder. We'll respond when we can.

## Not a support channel

GitHub issues in public Engine AI repos are not customer support. If you're a user of a client product built by Engine AI, contact that client directly.
